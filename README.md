## Course Modules (Main Topics)

- **Introduction & Basics**
  - Environment setup, program structure, Hello World, compilation
- **Syntax & Elements**
  - Comments, tokens, keywords, identifiers
- **Variables & Data Types**
  - Constants, literals, type conversions
- **Operators**
  - Arithmetic, relational, logical, bitwise, precedence
- **Control Flow**
  - Decision making, loops, break/continue, goto
- **Functions**
  - Prototypes, recursion, variadic functions, callbacks
- **Scope & Storage**
  - Global, static, linkage rules
- **Arrays & Strings**
  - Multi-dimensional arrays, dynamic arrays, string handling
- **Pointers**
  - Basics, pointer arithmetic, function pointers, pointers to structures
- **User-Defined Types**
  - Structs, unions, enums, typedef
- **Memory Management**
  - Layout, dynamic allocation, leaks
- **File Handling**
  - Input/output, operations, formatted output
- **Preprocessors**
  - Macros, pragmas, header files
- **Miscellaneous**
  - Error handling, math functions, random numbers, command line arguments
